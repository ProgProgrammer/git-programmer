
СОЗДАТЬ РЕГУЛЯРНОЕ ВЫРАЖЕНИЕ ДЛЯ ВСТАВКИ В ОДИН ИЗ ПУНКТОВ МЕНЮ ID ВМЕСТО ССЫЛКИ:

	Для этого нужно найти значение свойства Битрикс с символом "#" и, в зависимости от того, есть этот символ в строке или нет,
вывести на экран одну строку с html-тегами или другую. Если в строке есть символ "#", то перед выводом значения свойства на страницу,
удалить этот символ регулярным выражением. Для этого потребуется два регулярных выражения и один условный оператор.
Одно регулярное выражение выводит в условном операторе, есть ли нужный символ в строке или нет, а второе удаляет этот символ перед выводом на экран строки.

1) заходим в среду разработки;

2) вставляем и редактируем код:

<?php $regexp = "/#/ui";?>
<?php $match = [];?>
<?php $tur = preg_match($regexp, $arItem["LINK"], $match);?>
<?php $replacement = "$1";?>
<?php $id = preg_replace($regexp, $replacement, $arItem["LINK"]);?>
<?php if($tur == true) { ?>

<li class="nav-item"><span id="<?=$id?>"><?=$arItem["TEXT"]?></span></li>

<?php } else { ?>

<li class="nav-item"><a href="<?=$arItem["LINK"]?>"><?=$arItem["TEXT"]?></a></li>

<?php } ?>

3) <?php $regexp = "/#/ui";?> - шаблон регулярного выражения, где:
	# - символ, который нужно найти;
	u - выражение или текст использует кодировку utf-8;
	i - поиск ведется без учета регистров символов.

4) <?php $match = [];?> - массив, через который будет прогоняться строка;

5) $arItem["LINK"] - значение свойства Битрикс, в котором нужно найти совпадение;

6) <?php $tur = preg_match($regexp, $arItem["LINK"], $match);?> - регулярное выражение с присвоением, где:
	preg_match($regexp, $arItem["LINK"], $match) - регулярное выражение с переменными;
	$tur - переменная, которой присваивается результат, полученный регулярным выражением (0 или 1).

7) <?php $id = preg_replace($regexp, $replacement, $arItem["LINK"]);?> - регулярное выражение для замены совпадающей части строки, в котором содержатся:
	$regexp - шаблон. В данном случае это "/#/ui";
	$replacement - указание на место, в котором должна быть произведена замена. В данном случае "$1";
	$arItem["LINK"] - значение свойства Битрикс, в котором нужно найти совпадение.

8) <?php if($tur == true) { ?> - условный оператор, который выполняется, если переменная $tur равна 1 или true;

9) <?=$id?> - идентификатор, в котором символ "=" заменяет echo.
