ДЛЯ ОТПРАВКИ ПИСЬМА ПРИ ПОМОЩИ ФУНКЦИИ mail(), нужно сделать следующее:

1) вставить следующий код в файл send.php:

   <?php
    $message = '
                Имя: '.$_POST['name'].'
                Фамилия: '.$_POST['surname'].'
                Сообщение: '.$_POST['text'].'';
    
    mail("andreev@parksite.ru", 'Письмо из формы обратной связи сайта "CinemaOne"', "$message");
   ?>

2) можно воспользоваться всеми параметрами функции:

   mail(string $to , string $subject , string $message [, mixed $additional_headers [, string $additional_parameters ]]) : bool

3) можно посетить сайт с описанием функци mail: https://www.php.net/manual/ru/function.mail.php
