СОЗДАТЬ УСЛОВИЕ ОТОБРАЖЕНИЯ ЭЛЕМЕНТОВ В ЗАВИСИМОСТИ ОТ СТРАНИЦЫ:

	Так как у Joomla есть только один шаблон, то вносить условия нужно в один index.php.

1) зайти в среду разработки -> открыть файл index.php -> найти нужные блоки -> вставить код;

2) <?php
   $menu = &JSite::getMenu();
   if ($menu->getActive() != $menu->getDefault()) {
   ?>

   html-код

   <?php } ?>

   или

   <?php
   $menu = &JSite::getMenu();
   if ($menu->getActive() == $menu->getDefault()) {
   ?>

   html-код

   <?php } ?>

3) getDefault() - означает главную страницу. getActive() - обозначает активную страницу;

4) if ($menu->getActive() != $menu->getDefault()) - если активна не главная страница;
 
   if ($menu->getActive() == $menu->getDefault()) - если активна главная страница.





ИЗМЕНИТЬ СОДЕРЖАНИЕ СТРАНИЦЫ:

1) Зайти в админку Joomla;

2) в верхнем меню нажать на "Расширения" -> "Менеджер модулей";

3) выбрать нужный модуль -> открыть его, поправить и сохранить.






МЕСТОНАХОЖДЕНИЕ ШАБЛОНОВ САЙТА:

1) находятся в папке "templates" -> выбрать из папок нужный шаблон и открыть папку -> открыть папку "html" -> 

   открыть папку с шаблоном сайта;

2) если в шаблоне вместо html-кода существует вызов функции, то html-шаблон нужно искать в модулях (скорее всего в папке html).

   Возможно шаблон может быть в index.php шаблона сайта.






СОЗДАТЬ ССЫЛКУ ДО ФАЙЛА:

1) нужно прописать следующий путь: /templates/название шабона сайта/js/sendProduct.js;

2) при любом раскладе, находится сама папка "js" в папке "html" или нет, всегда прописывать этот путь.
