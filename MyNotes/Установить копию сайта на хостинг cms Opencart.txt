УСТАНОВКА КОПИИ САЙТА НА TIMEWEB С CMS OPENCART:

1) Зайти на сайт timeweb.ru -> Хостинг -> Виртуальный хостинг -> выбрать тариф Optimo+ -> Регистрация;

2) выбрать физическое лицо или юридическое лицо

2) заполнить все данные -> если есть купон, то нажать на "У меня есть код партнера" -> Заказать;

3) Зарегистрировать домен. Для этого войти в админке timeweb во вкладку "Домены и поддомены" -> ввести домен в input ->

   выбрать домен ru, com и т.д. -> Зарегистрировать -> заполнить все поля юридического или физического лица ->

   оплатить домен -> закончить регистрацию;

4) после того как домен был получен зайти во вкладку Сайты -> Создать новый сайт -> ввести Имя директории -> Создать

   удалить из public_html созданной директории сайта файл index.html и .htaccess, если имеется;

5) подключить домен. Для этого открыть настройки созданной директории сайта -> Настройки переадресации ->

   включить Перенаправлять на домен -> ввести домен без http -> Сохранить настройки;

6) подключиться к сайту по ftp -> закачать архивную копию сайта -> зайти во вкладку "Файловый менеджер" ->

   нажать на созданную директорию -> public_html -> нажать на архив -> Архиватор -> Разархивировать;

7) после разархивации перенести, а лучше скопировать разархивированные папки и файлы в корень сайта (в папку public_html),

   а также удалить созданный при копировании самим timeweb файл .htaccess (не путать с копированным .htaccess);

8) зайти во вкладку "Базы данных MySQL" -> Создание новой базы данных -> ввести новые логин и пароль -> Создать;

9) импортировать базу данных -> нажать на phpMyAdmin созданной базы данных -> выбрать домен сайта -> Импорт ->

   Выбрать файл -> Вперед;

10) создать в среде разработки новый проект этого сайта -> в корне сайта проверить формат файла ".htaccess". Если он указан
 
    не как ".htaccess", перевести его в этот формат (.htaccess.txt - недопустимо);

11) открыть файл config.php в корне сайта, а также в папке admin, которая находится в корне сайта,

    и вставить в строки файла пути от корня сайта до папки public_html везде, где указаны абсолютные ссылки на

    домен и относительные ссылки на папки;

12) формат относительной ссылки к корню сайта на timeweb: 

    /home/первая буква логина в личный кабинет на timeweb/логин в личный кабинет на timeweb/public_html/папка по умолчанию

    (уже указана);

13) формат абсолютной ссылки на домен: http://домен/ (например, http://cs30753.tmweb.ru/);

14) очистить кеш браузера -> открыть скопированный сайт.
