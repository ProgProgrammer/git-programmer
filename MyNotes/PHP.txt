
ВЫЗВАТЬ ФУНКЦИЮ ИЗ ДРУГОГО ФАЙЛА PHP В НУЖНЫЙ ФАЙЛ PHP:

1) зайти в файл php, из которого нужно вызвать функцию;

2) вставить код: <?php require_once('parseImage.php');?>, где require_once() - вызов файла php, а 'parseImage.php' -
  
   название файла.



ПРОВЕРКА СУЩЕСТВОВАНИЯ GET-ПАРАМЕТРА:

1) зайти в нужный файл php в среде разработки;

2) вставить код: isset($_GET['id']), где id - название GET-параметра (&id=123)




РАЗДЕЛИТЬ СТРОКУ ПО РАЗДЕЛИТЕЛЮ И СОЗДАТЬ МАССИВ СТРОК:

1) прописать метод explode();

2) $result = explode(".", "string") = где $result - массив полученных строк, "." - разделитель, а "string" - строка, которую разделяет метод.
