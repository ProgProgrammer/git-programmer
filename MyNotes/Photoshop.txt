
СКОПИРОВАТЬ НЕРАСТРОВЫЙ СЛОЙ:

1) выделить нужный элемент макета;

2) нажать правой кнопкой мыши по выделенному элементу во вкладке "Слои";

3) нажать на "Растрировать стиль слоя";

4) опять выделить нужный элемент во вкладке "Слои", если он не выделен и скопировать его при помощи cntrl+C;

5) в открывшейся закладке выделить во вкладке "Слои" верхний слой элемента -> правая кнопка мыши -> удалить слой;

6) файл -> Экспортировать -> Сохранить для Web.



ПРЕОБРАЗОВАТЬ СИМВОЛ В РУБЛИ:

1) выделить нужный символ текста

2) удалить в строке все символы, кроме нужного;

3) нажать правой кнопкой мыши по выделенному символу во вкладке "Слои" -> правая кнопка мыши -> Преобразовать в кривые;

4) нажать правой кнопкой мыши по выделенному символу во вкладке "Слои" -> Быстрый экспорт в SVG.




СКАЧАТЬ ФОН У МАКЕТА:

1) выделить нужную часть макета -> перейти на вкладку "Слои" -> сделать невидимыми все ненужные блоки;

2) нажать на Файл -> Экспортировать -> Быстрый экспорт в png или jpg;

3) есди нужно настроить экспорт, ниже Быстрого экспорта из 2-го пункта есть Настроить экспорт. В нем можно изменить параметры сжатия и т.д.
