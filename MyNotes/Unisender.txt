
https://www.unisender.com/ - сайт по рассылке почты.

СОЗДАТЬ ОТПРАВКУ ПОЧТЫ С САЙТА НА БИТРИКСе ЧЕРЕЗ UNISENDER НА СВОЮ ПОЧТУ:

1) зарегистрироваться и войти на сайт https://www.unisender.com/;

2) Нажать на Логин вверху справа -> Тарифы;

3) зайти на сайт https://www.unisender.com/ru/support/api/messages/sendemail/ -> изучить функцию sendMail и выбрать ее параметры;

4) в среде разработки выбрать файл init.php -> после кода определения почты сервиса на яндекс или другой почте -> 
 
   вставить после $config следующий код:

   function smtpmail($to = '', $mail_to, $subject, $message, $headers = '') { 
      $platform = 'My E-commerce product v1.0';
      $UnisenderApi = new UnisenderApi('6hgwwhna7ct5d3tk59wwct987jawpgobw6bzkp7a', 'UTF-8', 4, null, false, $platform);
      $UnisenderApi->sendEmail(['email' => $mail_to, 'sender_name' => 'Союз церковных фотографов', 'sender_email' => 'ufcppro@yandex.ru', 'subject' => $subject, 'body' => $message, 'list_id' => 19757581]);
   }		

   где:

   6hgwwhna7ct5d3tk59wwct987jawpgobw6bzkp7a - ключ API, который можно получить в Настройках аккаунта -> Интеграция и API -> Ключ доступа к API;
   функция или метод sendEmail([параметры функции]) - параметры записывать в порядке, указанном на странице https://www.unisender.com/ru/support/api/messages/sendemail/;
   list_id - можно узнать в админке сайта https://www.unisender.com/ (не помню как) либо при помощи метода getLists;
   $mail_to, $subject, $message - адрес отправки, тема письма и содержание письма взять из параметров родительской функции smtpmail;
   остальные значения параметров копировать из родительской функции;

5) сохранить файл init.php и проверить отправку письма.
