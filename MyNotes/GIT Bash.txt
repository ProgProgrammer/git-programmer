Использование ГИТа:

1) запустить Git Bash;

2) ввести команду после знака $: cd D:/Проекты/Синергия/site;

3) ввести команду после знака $: git clone https://bitbucket.org/bulletodds/halobactery-landing/src/master/

   Эта команда склонирует удаленный репозиторий. После этого запустить командную строку по правилам, указанным ниже для cmd (если сайт с js-фреймворком).

   Создать проект в IDE по правилам, указанным ниже;

4) открыть локальную копию проекта в браузере набрав, например, http://localhost:4200/;

5) после внесения исправлений в Гит Баш ввести команду: cd master - переключение на ветку "master";

6) сделать git add . , если добавлял новые или изменял действующие файлы (во всех случаях, когда git status показывает изменения красным цветом);

7) проверить количество модифицированных файлов командой git status;

8) сделать коммит git commit -am 'Правки верстки';

9) проверить количество незакоммичеснный файлов, если такие имеются, командой git status;

10) запушить файлы командой: git push origin master.
