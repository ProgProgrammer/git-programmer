ОТКЛЮЧИТЬ АТРИБУТ НА WORDPRESS:

1) зайти в админку wordpress;

2) Вкладка Товары -> Все товары;

3) Закладка Атрибуты -> Выбрать тип атрибута, например, Крепежная панель -> Удалить Да, оставить Нет;

4) Если в Атрибутах нет нужного, свойства, то создать и проделать дейтвия из пункта 3;

5) Зайти в вариации и поставить в пунктах Категории под вариациями -> Нет;

6) Обновить Wordpress.






ЗАМЕНИТЬ ФОН HEADER:

1) зайти в админку WordPress;

2) зайти в пункт Appearance (Вид);

3) нажать на Customize (Настройки);

4) в открывшимся окне зайти в Header (Заголовок);

5) нажать на Фон заголовка главной страницы;

6) указать в пункте Тип фона -> Image (Образ);

7) вставить в Заглавное изображение картинку.






ДОБАВИТЬ ИЛИ УДАЛИТЬ ЭЛЕМЕНТ СО СТРАНИЦЫ С КНОПКОЙ "ПОДРОБНЕЕ" У ЭЛЕМЕНТА

   Для этого нужно сначала добавить виджет "Подробнее", затем на странице добавить элемент на экран и
отредактировать его, прицепив к нему, при помощи id, содержимое виджета "Подробнее".

1) зайти в админку Wordpress -> Внешний вид -> Виджеты;

2) В пункте "Доступные виджеты" выбрать нужный тип виджета, например, HTML-код, + нажать на него -> выбрать нужную библиотеку

   виджетов, например, Toolbar Left, -> Добавить виджет;

3) внизу этой библиотеки появится виджет -> заполняем в нем Заголовок и Содержимое (содержимое в формате html-кода).
   
   В Содержимом нужно не забыть указать на родительском блоке id, чтобы вызвать виджет кнопкой "Подробнее". Например,

   id="modal-davidyan" davidyan - фамилия -> Сохранено -> Готово;

4) находим в админке в раздел "Страницы" -> находим нужную страницу и кликаем на нее -> нажать на кнопку
  
   "Редактор страниц". Этим действием мы перейдем на страницу с открытым окном редактора;

5) выбрать на странице в окне редактора нужную секцию -> выбрать свободный слот, если его нет, то создать пустой слот ->
  
   нажать на крест слота;

6) выбрать из всплывшего окна ярлык с названием "Panel" -> ввести заголовок -> контент на html. На кнопке "Подробнее"
   
   указать на вызов виджета по id. Например, href="#modal-davidyan" -> добавить изображение -> выставить ширину

   и высоту изображения -> указать alt;

7) зайти во вкладку "Настройки" -> все, что нужно задать. Например, в разделе "Изображение" можно выбрать форму

   отображаемого изображения и его тень;

8) во вкладе "Дополнительно" выставить ИМЯ ЭЛЕМЕНТА и нужные стили;

9) нажать на стрелку назад из элемента -> наверху конструктора нажать "Сохранить страницу";

10) зайти в админку Wordpress -> WP Fastest Cache -> вкладка Delete Cache -> кнопка Delete Cache;

11) если нужно УДАЛИТЬ ЭЛЕМЕНТ, то делаем то, что указано в пунктах 4 и 5. Затем, вместо добавления слота, удаляем нужный слот,

    нажав на корзину и повторяем действия, указанные в пунктах 9 и 10.






ОТРЕДАКТИРОВАТЬ МЕНЮ:

1) зайти в админку Wordpress;

2) раздел "Внешний вид" -> "Меню" -> "Перейти к визуальной настройке";

3) в левой панели выбрать нужное меню -> "Добавить элеметы" -> выбрать нужный элемент -> переместить пункт меню на нужную строку;

4) нажать на кнопку "Опубликовать";

5) повторить действия из пунктов 10 и 11 предыдущей главы.







ВНЕСТИ ИНФОРМАЦИЮ В ФОРМЫ:

1) войти в админку Wordpress -> Contact Form 7 -> Контактные формы;

2) выбрать нужную форму -> внести изменения -> Сохранить -> Сохранить(справа).






ИСПРАВИТЬ ТЕКСТ КУПОНОВ ИЛИ ДРУГИХ ПЛАГИНОВ:

В плагине строка находится по пути: Корень сайта -> wp-content -> plugins -> требумый плагин -> нужный файл.
Подробно про изменения языка с текстом написано здесь: https://wp-kama.ru/id_10492/perevody-v-wordpress.html#etap2
Подробности конкретного случая:

1) нужно войти на хостинг -> открыть файловый менеджер -> в поиске ввести строку, которую нужно заменить;

2) загрузить папку с найденным файлом (обычно располагается по пути Корень сайта -> wp-content -> plugins -> требумый плагин);

3) найти нужный файл со строкой для замены -> открыть php-файл со строкой -> заменить и сохранить.
